# Brick Breaker

### Description
A simple Brick Breaker clone made with p5.js

### Play
- https://cherrysoftmx.github.io/Brick-Breaker/

### Controls
| Key         | Action          |
| ----------  | ------------    |
| Left arrow  | Move to left    |
| Right Arrow | Move to right   |
| Spacebar    | Throw the ball  |
